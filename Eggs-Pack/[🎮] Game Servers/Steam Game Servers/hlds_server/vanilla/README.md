# HLDS Servers

This is for older games like CS 1.6 (default game) and other servers that still run on the older HLDS server under id 90

## Server Ports

HLDS servers require up to 6 ports

| Port      | default |
|-----------|---------|
| Game/rcon | 27015   |
| HLTV      | 27020   |
| VAC       | 26900   |
