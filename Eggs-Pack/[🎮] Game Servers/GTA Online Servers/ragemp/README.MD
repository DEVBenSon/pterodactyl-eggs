# Rage-MP

The [Rage-MP](https://rage.mp/) GTA 5 dedicated server
RAGE Multiplayer is an multiplayer modification for Grand Theft Auto V that is alternative to GTA Online.

## Server Ports

Warband requires a 2 ports (default 22005, 22006)

| Port    | default  |
|---------|----------|
| Game    | 22005    |
| http    | 22006    |
