# Tachyon

Tachyon is a multiplayer FTL server and client.

## Server Ports

The Tachyon server requires a single port for access.

| Port  | default |
|-------|---------|
| Game  | 30303   |

## Server Specific

Latest client and standalone server can be found at <https://github.com/Dav-Edward/TachyonsCE/releases>
